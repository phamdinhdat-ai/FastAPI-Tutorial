# Deep learning API

---

1. Create Conda Environment
    
    `conda create -n fastapi_ai python==3.10`
2. Clone Repo and install requirements
    - `https://github.com/phamdinhdat-ai/FastAPI-Tutorial.git`

    - `cd FastAPI-Tutorial/`

    - `pip install -r requirements.txt`
3. Run API 
    - `python server.py`
